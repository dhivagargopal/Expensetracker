# Personal Expense Tracker (CLI)

A simple, lightweight command-line expense tracker built in Python.  
It helps you log daily expenses, categorize them, and view summaries – all from your terminal.  
Perfect for anyone who wants a quick and offline way to keep track of spending without bloated apps.  

---

## Features
- Add expenses with amount, category, and optional notes  
- View summaries – total spending and category-wise breakdown  
- Export data to CSV for Excel/Google Sheets analysis  
- Persistent storage using JSON (your data is saved locally)  
- Simple, fast, and portable – runs anywhere Python does
